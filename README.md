# Portafolio Edinson Palacio

Un portafolio web moderno desarrollado con **Next.js**, **React**, **TypeScript** y **Tailwind CSS**. El proyecto presenta una interfaz limpia, responsiva y optimizada para mostrar información personal, habilidades, experiencia y proyectos de desarrollo.

## 🚀 Características

- Diseño moderno y atractivo.
- Interfaz completamente responsiva.
- Desarrollado con Next.js y React.
- Tipado estático mediante TypeScript.
- Estilos con Tailwind CSS.
- Componentes reutilizables.
- Optimizado para un buen rendimiento y experiencia de usuario.

## 🛠️ Tecnologías utilizadas

- Next.js
- React
- TypeScript
- Tailwind CSS
- CSS Modules
- ESLint


## ⚙️ Requisitos

Antes de ejecutar el proyecto asegúrate de tener instalado:

- Node.js 18 o superior
- npm

Puedes comprobar las versiones con:

```bash
node -v
npm -v
```

## 📥 Instalación

Clona el repositorio:

```bash
git clone https://github.com/Edinson629/Portafolio-Edinson.git
```

Ingresa al proyecto:

```bash
cd portfolio-edinson-palacio
```

Instala las dependencias:

```bash
npm install
```

## ▶️ Ejecutar el proyecto

Inicia el servidor de desarrollo:

```bash
npm run dev
```

Abre el navegador y visita:

```
http://localhost:3000
```

## 🏗️ Construcción para producción

Genera la versión optimizada:

```bash
npm run build
```

Ejecuta la aplicación en modo producción:

```bash
npm start
```
